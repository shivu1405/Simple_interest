# Simple Interest Calculator

This is a simple Python program to calculate Simple Interest.

## Formula Used

Simple Interest = (Principal × Rate × Time) / 100

## How It Works

The user enters:
- Principal amount
- Rate of interest
- Time period

The program calculates and displays the Simple Interest.

## Example

Input:
Principal = 1000  
Rate = 5  
Time = 2  

Output:
Simple Interest = 100.0
